How to create a animated grid

First create grid system

                
<div class="grid mt-5">
<div class="left"></div>
<div class="right"></div>
</div>

Then the css for the grid 

  
.left {
  background: lightblue;
}

.right {
  background: lightblue;
}

.grid {
  height: 50vh;
  background: lightcoral;
  gap: 3px;
  display: grid;
}

Now for the transition

There is a new class called .grid:has(.left:hover)

In plain English this is saying, “Do something to the .grid container if it contains an element named .left inside of it that is in a hover state.”

This allows for the grid transition to happen

We will also add an transition time and template to the grid

    
.grid:has(.left:hover) {
grid-template-columns: 50% auto;
}
.left:hover {
background-color: lightcoral;
}
.grid {
    height: 50vh;
    background: lightcoral;
    gap: 3px;
    transition: 400ms;
    display: grid;
    grid-template-columns: 48px auto;
}
